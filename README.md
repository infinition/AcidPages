# AcidPages

[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black) [![Release](https://img.shields.io/github/v/release/infinition/AcidPages?style=flat)](https://github.com/infinition/AcidPages/releases) [![Buy Me a Coffee](https://img.shields.io/badge/Buy%20Me%20a%20Coffee-ffdd00?style=flat&logo=buy-me-a-coffee&logoColor=black)](https://buymeacoffee.com/infinition)

A browser tool to scan any GitHub profile and list all its deployed GitHub Pages sites in one place.

Live: https://infinition.github.io/AcidPages/

<img width="1114" height="1038" alt="AcidPages interface" src="https://github.com/user-attachments/assets/0269320e-a420-49e6-a5ae-b9c04803258f" />

## Usage

Enter a GitHub username. AcidPages queries the GitHub API, finds every repository with Pages enabled, and lists the live URLs with metadata. No login or token needed for public profiles.

No installation required. Open `index.html` or use the live link above.

---

## Stack

- Vanilla JS + Tailwind CSS (CDN)
- GitHub REST API (public, unauthenticated)
